# 🎭 Automação de Testes com Playwright - SauceDemo

Este projeto faz parte da minha evolução como **QA Automation Engineer**. Aqui, realizo a automação de testes E2E (ponta a ponta) do site SauceDemo, utilizando o **Playwright**.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** JavaScript
* **Ferramenta de Teste:** Playwright
* **Versionamento:** Git & GitHub
* **Cenários de Teste:** Baseados em tickets (estilo Jira)

## 📋 Tickets Implementados (Sprint Atual)
- [x] **ADFDPW-01:** Login com usuário padrão (Caminho Feliz).
